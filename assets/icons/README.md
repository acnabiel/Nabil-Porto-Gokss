Letakkan logo resmi di folder ini dengan nama file berikut (format SVG/PNG):

- html5.svg
- css3.svg
- javascript.svg
- laravel.svg
- github.svg
- tailwindcss.svg

Tips:
- Gunakan versi resmi/logo dari situs resmi atau repositori Simple Icons.
- Prefer SVG supaya warna dan ukuran tetap tajam.
- Jika mau, ganti nama file di index.html sesuai file yang kamu taruh di sini.
